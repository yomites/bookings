# Bookings and Reservations

This is the repository for a booking and reservation project. 

- Built in Go version 1.25
- Uses the [chi router] (github.com/go-chi/chi) module for building Go HTTP services
- Uses [alex edwards SCS](github.com/alexedwards/scs/v2) session management module
- Uses the [nosurf](github.com/justinas/nosurf) module for preventing Cross-Site Request Forgery attacks
